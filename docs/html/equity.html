Equity

Equity market data, dividend curves, volatility surfaces, option instruments, and EQ pricing.

User Guide

Equity market data, dividend curves, volatility surfaces, option instruments, and EQ pricing. This page contains 36 functions. Use the filter to narrow by function name, return type, parameter, or description.

Example

eqRisk = caplib::CreateEqRiskSettings(0, 0, 0, 0, "EQ_NO_RISK", true);
eqMkt = caplib::CreateEqMktDataSet(
    2025.01.02, "EQ_UNDERLYING", 100.0,
    "USD_DISC", "EQ_DIV", "EQ_VOL", "", "", 0.0, "EQ_MKT", true);

pricingResult = caplib::PriceEqEuropeanOption(
    "EO_INST", 2025.01.02, eqMkt[0], "BSM_ANALYTICAL", eqRisk[0], "", "", true);
resultJson = pricingResult[1];
