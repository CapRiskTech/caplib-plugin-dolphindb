User Guide

1. Create objects

Use factory functions to create curves, quotes, templates, settings, market data, and instruments in ObjectCache.

2. Pass handles

Builder and pricing wrappers take handle strings from previous calls instead of repeated serialized protobuf bytes.

3. Price consistently

Native pricing wrappers use the same seven data arguments, optionally followed by returnJson BOOL: instrument, pricing date, market data, pricing settings, risk settings, scenario settings, mode, and returnJson.

4. Visualize JSON

Omit returnJson or set it to false for handle-only output; set it to true for [handle, protobufJson] inspection output.

Return pattern

pricingResult = caplib::PriceEqEuropeanOption(inst, date, mkt, settings, risk, "", "", true)
pricingHandle = pricingResult[0]
pricingJson = pricingResult[1]
