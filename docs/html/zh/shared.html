DolphinDB caplib 插件

通用

跨资产工具、静态数据服务辅助函数、定价设置、风险设置、情景设置以及共享 protobuf 工具。

用户指南

跨资产工具、静态数据服务辅助函数、定价设置、风险设置、情景设置以及共享 protobuf 工具。 本页包含 21 个函数。可按函数名、返回类型、参数或说明进行筛选。

所有会向 内存对象 写入对象的函数都支持可选的最后一个参数 returnJson BOOL。省略或传入 false 时仅返回句柄；传入 true 时返回 [handle, protobufJson]。

示例

settings = caplib::CreatePricingSettings(
    "USD", "ANALYTICAL", 0, 0,
    "BSM_MODEL", "", "", "BSM_ANALYTICAL", false);

json = caplib::GetObjectCacheJson("PricingSettings", settings);
