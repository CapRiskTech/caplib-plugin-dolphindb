用户指南

1. 创建对象

使用工厂函数在 内存对象 中创建曲线、报价、模板、设置、市场数据和工具对象。

2. 传递句柄

构建器和定价封装函数接收前序调用返回的句柄字符串，避免重复传递序列化 protobuf 字节。

3. 统一定价调用

原生定价封装使用一致的七个数据参数，并可追加 returnJson BOOL：工具、定价日、市场数据、定价设置、风险设置、情景设置、模式以及 returnJson。

4. 查看 JSON

省略 returnJson 或设置为 false 时仅返回句柄；设置为 true 时返回 [handle, protobufJson]，便于检查和可视化。

返回值模式

pricingResult = caplib::PriceEqEuropeanOption(inst, date, mkt, settings, risk, "", "", true)
pricingHandle = pricingResult[0]
pricingJson = pricingResult[1]
