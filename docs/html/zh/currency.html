用户指南

外汇计算器、报价矩阵、波动率曲面、现金类产品和期权定价。 本页包含 39 个函数。可按函数名、返回类型、参数或说明进行筛选。

示例

fxMkt = caplib::CreateFxMktDataSet(
    2025.01.02, "USD_DISC", "EUR_DISC", "EURUSD_SPOT", "EURUSD_VOL", "FX_MKT", "FX_MKT", true);

pricingResult = caplib::PriceFxEuropeanOption(
    "EURUSD_CALL", 2025.01.02, fxMkt[0], "FX_ANALYTICAL", "FX_RISK", "", "", true);
pricingHandle = pricingResult[0];
pricingJson = pricingResult[1];
