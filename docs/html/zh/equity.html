用户指南

权益市场数据、股息曲线、波动率曲面、期权工具和权益定价。 本页包含 36 个函数。可按函数名、返回类型、参数或说明进行筛选。

示例

eqRisk = caplib::CreateEqRiskSettings(0, 0, 0, 0, "EQ_NO_RISK", true);
eqMkt = caplib::CreateEqMktDataSet(
    2025.01.02, "EQ_UNDERLYING", 100.0,
    "USD_DISC", "EQ_DIV", "EQ_VOL", "", "", 0.0, "EQ_MKT", true);

pricingResult = caplib::PriceEqEuropeanOption(
    "EO_INST", 2025.01.02, eqMkt[0], "BSM_ANALYTICAL", eqRisk[0], "", "", true);
resultJson = pricingResult[1];
