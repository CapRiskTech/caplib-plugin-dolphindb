User Guide

Cross-asset utilities, static-data service helpers, pricing settings, risk settings, scenario settings, and shared protobuf tooling. This page contains 21 functions. Use the filter to narrow by function name, return type, parameter, or description.

Every function that stores an object in ObjectCache accepts an optional final returnJson BOOL argument. If omitted or false, the function returns only the handle. Pass true to return [handle, protobufJson].

Example

settings = caplib::CreatePricingSettings(
    "USD", "ANALYTICAL", 0, 0,
    "BSM_MODEL", "", "", "BSM_ANALYTICAL", false);

json = caplib::GetObjectCacheJson("PricingSettings", settings);
